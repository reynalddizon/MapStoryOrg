Geographic Attributes

Help MapStory determine the geographic information in your dataset.


Map Projection

Map projection is the process on which a round, 3D surface of the earth is represented on a flat two-dimensional map. By default, all maps are projected using WGS 84 projection. Read more about Map Projections.

Projection: WGS 84

Latitude

Latitude is the the distance of a point north or south from the equator. Values range from 0 to 90 degrees for North values, and from -90 to 0 degrees for South values. All values must be in decimal degrees (dd.xxxxx). Read more about latitude.

Longitude

Longitude is the the distance of a point east or west from the prime meridian. Values range from 0 to 180 degrees for East values, and from -180 to 0 degrees for West values. All values must be in decimal degrees (dd.xxxxx). Read more about longitude.

Simple: Time column and format

/*this appears only when the Simple option is selected. This should be similar to the start time on the current production*/

Specify the column that contains the time information of the dataset. If the column is in number format, values are automatically translated as year values. If the column is in text format, you need to specify the date format. Learn more about Time formats.

Tips on date formats:

  • MM/dd/yyyy
  • M/d/y
  • MM/dd/yyyy'T'HH:mm:ss
  • MM/dd/yyyy'T'hh:mm:ss
